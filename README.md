# 🚀 SkillNest - AI-Based Smart Internship Verification and Performance Analytics Platform

> 🎯 Deployed Full-Stack MERN Application using Render & Vercel

---

## 📌 Overview

SkillNest is an AI-powered internship platform designed to help students discover, manage, and analyze internship opportunities efficiently.
It provides a smart dashboard with recommendations and performance insights.

## ✨ Features

* 🔐 User Authentication (Login / Register)
* 📚 Internship Listings
* 🤖 AI-Based Recommendations
* 📊 Dashboard Analytics
* 🧑‍💻 Admin Controls (optional)

## 🛠️ Tech Stack

**Frontend:**

* React (Vite)
* Tailwind CSS

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB Atlas

---

## ⚙️ Installation & Setup

### 🔧 Backend

```bash
cd skillnest-backend
npm install
npm start
```

---

### 🎨 Frontend

```bash
cd skillnest-frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside backend folder:

```
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```
